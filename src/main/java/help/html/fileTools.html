Tools:

  • New File

    New file tool allows you to edit a blank text area. If you have made any changes on the actual file and click on it, it will ask you if you want to save the file. If you have not made any changes it will open a new blank text area.

    You can find new file tool on the top left menu item "File", on the left buttons with an icon in the shape of a paper and a plus symbol.

    You can also use the Ctrl+N combination to use it.

  • Open File

    Open file tool allows you to edit a selected plain text file. If you have made any changes on the actual file and click on it, it will ask you if you want to save the file. If you have not made any changes it will open a file explorer so you can select a plain text file to edit.

    You can find open file tool on the top left menu item "File", on the left buttons with an icon in the shape of a half opened folder.

    You can also use the Ctrl+O combination to use it.

  • Save

    Save saves the file that is being displayed at the moment of the interaction. If you have made any changes on the actual file and click on it, it will save the changes on that file. If the file does not exist it will open a file explorer so you can select the path where you want to save the text file.

    You can find Save tool on the top left menu item "Edit", on the left buttons with an icon in the shape of a dark floppy disk.

    You can also use the Ctrl+S combination to use it.

  • Save as

    Save as opens the file explorer so you can save the file that is being displayed at the moment of the interaction. You can select the path where you want to save the text file. If the file already exist in the selected location, the program will ask if you want to overwrite it.

    You can find Save as tool on the top left menu item "Edit", on the left buttons with an icon in the shape of a light floppy disk.

    You can also use the Ctrl+Shift+S combination to use it.
