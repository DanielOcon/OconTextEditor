Welcome to My Text Editor Program!

This text editor is a simple yet powerful tool for editing plain text files.

Features:

  • Open and save plain text files with ease.
  • Choose from a selection of fonts and font sizes.
  • Select text color using a color picker.
  • Undo and redo changes to your document.
  • Cut, copy and paste tools.

Usage:

To use the text editor, simply open a plain text file by clicking on "New file" or the "Open" button and navigating to the file location. Once the file is open, you can edit the text as desired. Use the "Save" button to save your changes to the file.

Use the font selection dropdown and font size slider to adjust the appearance of the text. Use the color picker to change the color of selected text.

New file, open file, save, save as explained on "File tools" window.

Undo, redo, cut, copy and paste tools explained on "Editing tools" window.
