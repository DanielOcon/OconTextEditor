Editing Tools

Here is an explanation on how to work with the different tools and where to find them.

Tools:

  • Cut

    Cut tool deletes the selected text from the text area and saves it on the clipboard. If you copy or cut another text it will be overwritten, so be careful.

    You can find cut tool on the top left menu item "Edit", on the center-ish buttons with an icon in the shape of scissors or right clicking on the document.

    You can also use the Ctrl+X combination to use it.

  • Copy

    Copy tool copies the selected text from the text area and saves it on the clipboard. If you copy or cut another text it will be overwritten, so be careful.

    You can find copy tool on the top left menu item "Edit", on the center-ish buttons with an icon in the shape of two papes overlayed or right clicking on the document.

    You can also use the Ctrl+C combination to use it.

  • Paste

    Paste tool pastes the text from the clipboard. If you copy or cut another text it will be overwritten on the clipboard and the tool will paste the new text, so be careful.

    You can find paste tool on the top left menu item "Edit", on the center-ish buttons with an icon in the shape of a paper on a clipboard or right clicking on the document.

    You can also use the Ctrl+V combination to use it.

  • Undo

    Undo goes back to the previous state of the text. If you do any changes on any undo state you will not be able to redo the state again, so be careful.

    You can find undo tool on the top left menu item "Edit", on the center-ish buttons with an icon in the shape of an arrow pointing left or right clicking on the document.

    You can also use the Ctrl+Z combination to use it.

  • Redo

    Redo goes foroward to the previous state of the undo tool. If you do any changes on any undo state you will not be able to redo the state again, so be careful.

    You can find redo tool on the top left menu item "Edit", on the center-ish buttons with an icon in the shape of an arrow pointing right or right clicking on the document.

    You can also use the Ctrl+Y combination to use it.
